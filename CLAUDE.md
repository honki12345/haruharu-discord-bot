# CLAUDE.md

이 파일은 Claude Code가 프로젝트를 이해하고 작업할 때 참고하는 가이드입니다.

## 문서 참고 가이드

### @docs/PROJECT.md

다음 경우에 참고:

- 프로젝트 구조를 파악할 때
- 슬래시 커맨드 목록과 파라미터를 확인할 때
- 이벤트 핸들러 동작 방식을 이해할 때
- 데이터베이스 모델(테이블) 스키마를 확인할 때
- 유틸리티 함수나 상수를 확인할 때
- config.json 설정 항목을 확인할 때

### @docs/USER_STORIES.md

다음 경우에 참고:

- 새로운 기능을 구현할 때
- 기존 기능의 비즈니스 로직을 이해할 때
- 시퀀스 다이어그램으로 흐름을 파악할 때
- 인수 조건(acceptance criteria)을 확인할 때

### @docs/COMMIT_CONVENTION.md

다음 경우에 참고:

- 커밋 메시지를 작성할 때
- 커밋 타입(feat, fix, docs 등)을 결정할 때
- 커밋 전 검증 절차를 확인할 때

### @docs/BRANCH_CONVENTION.md

다음 경우에 참고:

- 새 브랜치를 생성할 때
- 브랜치 이름 형식을 확인할 때

### @docs/pull_request_template.md

다음 경우에 참고:

- PR을 생성할 때
- PR 본문 형식을 확인할 때

### @docs/HOLIDAY_POLICY.md

다음 경우에 참고:

- 공휴일 목록을 업데이트할 때
- 새해 공휴일을 추가할 때
- 공휴일 관련 코드를 수정할 때
